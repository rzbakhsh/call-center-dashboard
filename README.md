# لوحة تحكم أداء موظفين مركز الاتصال
**Call Center Performance Dashboard — إمداد & تمهير**

---

## 🚀 طريقة النشر على GitHub Pages

### المرة الأولى فقط:
1. افتح [github.com](https://github.com) وسجّل دخول
2. اضغط **New repository** → اسمه مثلاً `cc-dashboard`
3. ارفع الملفين: `index.html` و `data.json`
4. اذهب إلى **Settings → Pages → Branch: main → Save**
5. الرابط سيكون: `https://USERNAME.github.io/cc-dashboard`

---

## 📊 طريقة تحديث البيانات

**فقط عدّل `data.json` — لا تلمس `index.html`**

### هيكل السجل في data.json:
```json
{
  "n": "اسم الموظف",
  "l": "J",          ← J = جدة, M = مكة
  "d": "30-مايو",    ← التاريخ
  "s": 1,            ← 1 = حاضر, 0 = غائب
  "wh": 10.02,       ← ساعات العمل
  "sm": 0,           ← دقائق التقصير
  "c": 32,           ← عدد المكالمات
  "cs": 1.0,         ← CSAT (0 إلى 1)
  "sc": 0.95,        ← الدرجة النهائية (0 إلى 1)
  "st": 1,           ← 1 = نوبة 10h, 0 = نوبة 8h
  "ah": 2.3,         ← متوسط مدة المكالمة بالدقائق
  "ce": 5,           ← عدد المكالمات المُقيَّمة
  "sv": "اسم المشرف",
  "nt": "ملاحظات",
  "ms": ""           ← بيانات ناقصة (يُترك فارغاً عادةً)
}
```

### لإضافة يوم جديد:
1. افتح `data.json`
2. أضف التاريخ الجديد في مصفوفة `"dates"`
3. أضف السجلات الجديدة في مصفوفة `"records"`
4. ارفع الملف المحدّث على GitHub → يتحدث الموقع تلقائياً
